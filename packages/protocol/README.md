## ZKP Service

A x402 service montizing zk proof generation for agents and agents registered acc to ERC-8004 standard for reputation and validation



### Development

#### Build

```
npx hardhat build
```

#### Test

```
npx hardhat test solidity
```

#### Coverage

```
npx hardhat test solidity --coverage
```
